# 末世虫潮 · DNA 卡

## 一句话定位
末世爽文：靠吞噬进化与信息差碾压，一路收后宫建势力。

## 戏剧结构
- **整书基调**：轻松吐槽风末世爽文。用大量口语、动漫梗和内心独白消解末世的沉重感，核心是“主角带着家人兄弟一路碾压变强”，而非生存挣扎。
- **典型的“5 章情节单元”长什么样**：
  - **起（1 章）**：确立新目标或解决旧敌。目标明确、可量化（如“帮我杀十只变异生物”、“速通黑暗研究室”），主角主动出击。
  - **承（1-2 章）**：执行任务，遭遇意外。任务难度远超预期（如蜂后变虫族储兵点、晶甲虫数量庞大），信息量密集，通过战斗和对话推进。
  - **转（1 章）**：核心爽点释放。主角用伏笔、智力或绝对实力完成反杀/破局（如生寄虫卵反杀蜂后、嘴炮忽悠超梦），同时抛出新的危机或任务（如虫族三月后总攻、发现晶甲虫隐患）。
  - **合（1 章）**：战利品消化与新计划预告。测试新能力/装备，安顿新角色，明确下一阶段行动目标（如猎杀音速蜂鸟、培养新徒弟）。
- **信息释放节奏**：
  - **即时满足**：新能力、新装备获得后立刻展示（如获得“凝水成兵”后马上制造水巨人）。
  - **前置设定**：进入新场景前，通过角色对话或旁白预先交代规则（如地狱每层的惩罚规则）。
  - **信息错位**：频繁使用“读者先于主角知道”的手法。通过切换反派视角或上帝视角，让读者知道危机正在逼近，而主角毫不知情，制造紧张感和优越感（如读者知道河豚肥婆要背叛，水母姬不知；读者知道火神教绑架白小玩，古月不知）。
  - **长线悬念**：核心危机提前预告（如“三个月后虫族总攻”），主角的某些身份秘密（如孙尚失忆前身份）和终极敌人（如“藏在暗处的强敌”）被刻意保留。
- **爽点频率**：
  - **小爽点（每 1-2 章）**：实力碾压杂兵、收服新角色/宠物、获得新装备/能力、利用情报优势掌控局面。
  - **大爽点（每 5 章单元结尾）**：用伏笔或智力完成绝境反杀、神器吞噬进化、强者回归清扫战场、绝境中发现希望反转（如白小玩之死可逆）。

## 写作风格
- **叙述视角 + POV 策略**：以主角古月的有限第三人称视角为主，但频繁切换到配角、反派甚至路人视角，形成“跟拍式”多视角。偶尔插入上帝视角解释设定或反派动机，制造信息错位。叙述者还会跳出代表“观众/读者”吐槽，打破第四面墙。
- **句式 / 节奏 / 对白密度**：
  - **句式**：以短句和中等长度句子为主，动作和决断时多用短句，干脆利落。
  - **对白密度**：极高。对白是推动情节、交代设定、塑造性格的主要手段，占比约 60%-70%。角色靠对话和吐槽确立人设。
  - **节奏**：呈现“战斗-对话-战斗”的交替节奏。战斗描写快，多用短句和动作堆叠；对话和设定解释时放缓。
- **特色修辞 / 高频词 / 作者口味**：
  - **修辞**：大量使用网络流行语、动漫游戏梗（如“海贼王的技能”、“你以为你是佐助啊”）和夸张吐槽，风格轻松“宅”向。
  - **高频词/句式**：“靠”、“尼玛”、“坑爹啊”、“……”、“（内心独白式吐槽）”。
  - **作者口味**：偏爱用数字量化实力（“战斗力三十万”、“十亿兵蜂”、“千分之一的实力”），用“反差萌”塑造角色（胆小吐槽役+三无数据娘），用“先抑后扬”制造爽点。
- **章末钩子的三种主力款式**：
  1. **威胁升级型**：发现敌人数量/实力远超预期，或将个人小事升级为生存危机（如“晶甲虫隐患”、“三个月后虫族总攻”）。
  2. **未完战斗/新任务预告型**：战斗在关键时刻戛然而止，或明确下一章的行动目标（如“天劫之雷 vs 制裁之雷”、“猎杀音速蜂鸟”）。
  3. **新角色入场/谜底浮现型**：极具冲击力的新角色登场并留下悬念，或揭示一个令人震惊的真相（如潘多拉破腹而出称古月“父神”、超梦被嘴炮忽悠瘸）。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：守护自己的逍遥生活与身边人。一切行动（建营地、下冥界、结束战争）都围绕此展开，利己但不冷血。
  - **性格关键词**：戏谑吐槽、杀伐果断、护短、利己主义、喜欢用智力而非蛮力解决问题。
- **主角的金手指是什么 + 作者如何避免主角“开挂无趣”**：
  - **金手指**：吞噬进化（夺取生物能力）、演神力/第六感（预知、推算）、专属武器（葬神戟，可吞噬神器进化）、小世界（可收人，并潜移默化影响忠诚）。
  - **避免无趣的方式**：
    1. **伏笔回收**：看似不起眼的早期能力（如生寄虫卵）在关键时刻成为破局杀招，而非临时开挂。
    2. **智力碾压**：主角常利用信息差、敌人设定漏洞或战术谋划取胜（如用“精神病人思维”忽悠超梦）。
    3. **能力限制**：新能力有明确消耗或限制（如“绝对催眠”消耗巨大），并非无脑使用。
    4. **强者对比**：通过“强者视角”与“普通人视角”的切换，侧面烘托主角强大，而非一味正面描写。
- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型形象**：残忍、有明确动机（复活洛基、报复人类）、会主动策划阴谋（绑架、背叛）的极端分子。
  - **威胁值维持方式**：
    1. **造成实质性伤害**：反派能成功绑架核心角色（白小玩）、屠杀小镇、重伤主角队友。
    2. **能力逐步揭露**：反派能力层层递进，从物理攻击到空间攻击再到新形态，不断刷新强度。
    3. **信息错位**：读者先于主角知道反派的阴谋和行动，看着危机逼近，增强焦虑感。
- **本书价值观锚点**：读者被什么打动？
  - **爽感（绝对碾压）**：主角用绝对实力或智力轻松解决看似无解的危机。
  - **成长（养成感）**：收徒弟、培养势力（潘多拉生产大军）、神器进化。
  - **友情/羁绊**：核心团队间的信任和吐槽，为守护家人不惜下冥界。
  - **荒诞幽默**：用轻松吐槽和意外反转（嘴炮通关）消解末世沉重感。

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1. 末世由虫潮引发，虫族有组织（蜂皇储兵点）、有总攻计划（三个月后）。
  2. 存在游戏化的能力体系：新人类、强化系、精神系、个性武器、军功系统。
  3. 存在多元神话体系：北欧神话（奥丁、昆古尼尔）、东方地府（十八层地狱、十殿阎罗）、希腊神话（雅典娜）等共存。
  4. 灵魂不泯灭，可通过祭献进入冥界，有复活可能。
  5. 存在“小世界”空间，可收容生物，并对其施加忠诚印记。
- **主角专属设定**：
  - **金手指来源**：吞噬能力，可吞噬生物并夺取其能力。
  - **身份秘密**：拥有演神力，可推算因果；拥有专属武器“葬神戟”，需吞噬神器进化。
  - **真实立场**：只想守护身边人过逍遥日子，被迫卷入种族战争。
- **专属角色 / 组织 / 地名（前 5 个）**：
  1. **古月**：主角。
  2. **夏舒**：亚洲最强新人类，拥有“绝对领域”。
  3. **纽克斯**：幻想生物，与丫头有神秘联系，恐惧被“另一个自己”吞噬。
  4. **潘多拉**：由生寄虫卵孵化，称古月为“父神”，能生产兵蜂大军。
  5. **不灭城**：人类最大聚集地。
- **专属物品 / 技能 / 概念名词（前 5 个）**：
  1. **葬神戟**：主角专属武器，可吞噬神器进化。
  2. **生寄虫卵**：可寄生并控制/吞噬宿主，孵化出潘多拉。
  3. **回光返照/目标档案**：演神力技能，可查看目标信息和过去。
  4. **绝对领域**：夏舒的能力，领域内自称神。
  5. **冥界之门**：黑魔法，可打开通往冥界的通道。

## 可借鉴的操作手法
- **情节层**：
  1. **量化任务目标**：将角色动机转化为具体、可量化的行动（如“杀十只变异生物”），让情节推进有明确的进度条。
  2. **伏笔回收式破局**：让主角用很早之前获得的、看似不起眼的能力或物品，作为当前绝境的关键解法，制造“原来如此”的爽感。
  3. **绝境中的希望反转**：在看似无可挽回的打击（如角色死亡）后，立刻通过设定揭示其可逆性（如灵魂不灭可复活），将压抑瞬间转为新希望。
  4. **反派视角铺垫危机**：切换反派视角，详细描写其阴谋策划过程，让读者先于主角知道危机全貌，制造紧张感和期待感。
  5. **“嘴炮”或智力通关**：在读者预期会有一场苦战时，用角色设定漏洞或逻辑诡辩“说服”敌人，制造强烈的意外感和智力爽感。
- **风格层**：
  1. **吐槽式叙述消解沉重**：在严肃或紧张的情节中，插入角色间的吐槽、内心独白或现代梗，调节叙事节奏，形成轻松阅读感。
  2. **“强者视角”对比法**：先描写主角在极短时间内完成复杂战斗，再切回普通人视角展示“无事发生”，用对比侧面烘托强大。
  3. **反差萌人设标签化**：给角色贴上对立性格标签（如“胆小吐槽役”+“三无数据娘”），并通过对话快速强化，让角色一出场就立住。
  4. **数字量化实力**：用具体数字（战斗力、数量、百分比）描述实力差距或危机规模，让读者有直观认知。
  5. **打破第四面墙**：叙述者偶尔跳出，代表“观众/读者”进行吐槽，增强互动感和网文特有的轻松氛围。
- **人物层**：
  1. **“恩情契约”绑定强力盟友**：让主角通过救命之恩，与远强于自己的角色建立临时契约关系（如“帮我杀十只怪”），合理利用高端战力而不破坏平衡。
  2. **工具型角色快速定性**：新角色出场即通过功能（提供线索、充当向导）和标签化性格（好色吝啬、豪爽粗线条）快速定性，用完即走或收编。
  3. **情感铺垫+爆发式觉醒**：为配角安排温馨日常建立读者情感，再用惨烈事件引爆其仇恨，完成从弱者到强者的戏剧性转变，开启新支线。
  4. **“先闻其声”的铺垫式出场**：重要角色出场前，先通过其他角色的恐惧、议论或环境氛围进行铺垫，再正式亮相，增强气场。
  5. **用切磋战展示团队关系**：通过一场非致命的团队内部切磋，一次性展示多个配角的性格、能力和彼此关系，效率极高。

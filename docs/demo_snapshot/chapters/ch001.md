# 第一章 · 九龙城寨的第一顿饭

六月的九龙城寨，地下水沟的味道和烧腊香混在一起，林家耀分辨不出哪种更重。

他站在东头村道的骑楼底下，背心黏在后背上，汗干了又湿。罗湖桥上那趟火车把他从对岸拉过来，铁栏杆被烈日晒得烫手，掌心贴上去能闻到铁锈和汗混合的微腥。那是四个钟头前的事。

口袋里的港币还剩十二块七。

表叔没在家。

林家耀按了三遍门铃，出来开门的是个赤膊阿伯，满嘴黄牙，说陈记一家上个月搬去观塘工厂区，地址没留。他道了谢，转身下楼时听见阿伯在后面骂了句“北边来的蝗虫”。

骑楼底下有家冰室，吊扇哗哗转，玻璃柜里摆着蛋挞和菠萝包。林家耀在门口站了五秒，看一眼价钱牌，转身走开。

十二块七不够住一晚时钟酒店。不够买一张回罗湖的车票。够吃三碗云吞面，但他不知道下一顿在哪。

他把手插进裤袋，摸到烟盒，空的。

城寨的巷道像被谁随手捏出来的，头顶是密密麻麻的冷气机和晾衣竹竿，水珠滴在帆布棚上，啪嗒声像断续的秒针。一条巷子拐进去，左右都是无牌诊所和牙医档口，招牌上的红漆字褪成粉色。再往里走，闻到熬中药的苦味混着烧焊的金属气。

林家耀在一条巷子尽头停下，靠墙蹲下。膝盖抵着胸口，能听见自己胃在响。

他闭上眼。

脑子里跳出那块半透明的面板，蓝底白字，像老式电脑屏幕。

【港务档案】系统。

他前世在2024年死的——车祸，不痛，快到来不及怕。醒来人已经在罗湖桥那头，手里攥着张单程证，证件上的照片是他自己，名字也是。二十二岁，福建泉州人，投靠表叔陈国良。

系统是跟着他一起醒的。

他试过几回。脑子里想一个名字、一个日期，屏幕上就跳出对应的公开资料。报纸头条、股价走势、法庭判决、台风路径。但仅限1983年到2000年之间公开过的东西。私人的查不到，没公开的查不到。

初始一百点情报值。查一次扣一到五点，看信息量。

林家耀睁开眼，肚子又叫一声。

饿不是最麻烦的。麻烦的是今晚住哪。九龙城寨的时钟酒店一晚十五块，他不够。天桥底？差佬半夜巡街会查身份证。他这张单程证刚入境，连工作签证都没换。

他站起身，膝盖咔嚓响了一声。

走出巷子，东头村道口有家大排档，折叠桌摆到骑楼外，铁锅颠勺时窜起的火舌舔着黑夜。油爆声和食客的喧哗混在一起，镬气裹着蒜蓉香扑过来。

林家耀在骑楼对面站定。

大排档门口三个年轻人在围一个穿校服的男生。

男生瘦小，校服口袋里插着支圆珠笔，背带裤的肩带被扯断一根，垂在腿边晃。三个年轻人年纪不大，顶多二十出头，花衬衫塞进高腰牛仔裤，其中一个剃板寸，脖子上挂条假金链。

“阿威，今个月嘅保护费呢？”板寸头推男生肩膀。

阿威后退一步，后背撞上折叠桌，桌上的辣椒酱瓶晃了晃。

“我冇钱。”他声音细，像蚊子叫。

“冇钱？”板寸头笑出声，“你老母唔系有肺痨咩？冇钱睇医生？”

阿威低着头，手指揪住校服下摆。

板寸头伸手扯他口袋。校服口袋是缝上去的薄布片，嘶啦一声裂开，几个硬币滚出来，叮叮当当掉在地上。有个五毫子滚进路边的污水沟，发出沉闷的叮一声。

三个混混一起笑。

阿威蹲下去捡，板寸一脚踩住他的手。“想捡？嗌声大佬先。”

林家耀看着这一幕。心里算了笔账。

救下这个本地仔，今晚就有地方睡，明天就有人带路找黑工。这小子是城寨地头蛇，母亲住院欠着医药费，正是用钱能拴住的人。三个散仔而已，打完就跑，只要不在现场留名，十四K不会为一个底层混混翻遍城寨。

他需要一个人。一个能在城寨给他指路的人。

林家耀在脑子里打开系统。

输入：九龙城寨东头村道大排档 老板 1983

屏幕闪了闪。

【查询消耗情报值 3 点，剩余 97 点】

【结果：赵广财，1955年由潮州经澳门偷渡入港，1973年购入东头村道37号地铺经营大排档。1981年次子赵志伟骑摩托车撞伤孕妇后逃逸，由十四K城寨坐馆出面摆平，欠下坐馆三万八千元人情债。1984年3月还清。】

林家耀关掉屏幕。

够了。

他走过去。

三个混混正揪着阿威的衣领，把他往巷子里拖。阿威两只手抓住折叠桌腿，指节发白。食客都低下头，筷子照样动，眼睛盯着自己碗里的面。

“放手。”林家耀说。

板寸头转头，上下打量他。

“乜水？”花衬衫带点潮州口音。

“佢系我细佬。”

阿威抬头看他，眼睛里全是疑惑，但嘴巴闭得死紧。

板寸头松开阿威，朝他走两步。另外两个从左右包过来。

“你边度㗎？”板寸头问，“睇你样唔似城寨人。”

林家耀没答，侧头看一眼大排档里面。灶台后面的老板正翻锅，左手抖勺右手颠锅，油爆声盖住外面的动静。一个四十来岁的女人在收银台数零钞，应该是老板娘。

“你哋三个，”林家耀说，“即刻走。”

板寸头愣了半秒，然后笑了。“凭乜？”

林家耀没看他，提高声音朝大排档里面叫了一声。

“赵广财！”

灶台后面的老板手一抖，铁锅差点脱手。

他转过头，五十岁左右，方脸，左边眉骨有块烫伤的旧疤。围裙上全是油渍，手里还握着铁勺。

林家耀看着他，声音很轻，轻到只有老板能听见。

“赵志伟嘅嘢，你唔想我喺度讲。”

老板的脸在一瞬间变了。

不是害怕。是那种被人从背后捅了一刀才有的表情——震惊、不信，然后迅速压下去，换成一张笑脸。他放下铁勺走出来，在围裙上擦擦手。头顶昏黄灯泡晃了晃，飞蛾影子在他脸上掠过。

“几位兄弟，俾面俾面，”老板笑着朝三个混混拱手，“有乜嘢慢慢讲，唔好喺我档口搞事。”

板寸头皱眉。“财叔，我哋收数啫——”

“收数去第二度收，”老板的语气还是笑的，但手里不知什么时候多了把菜刀，刀刃朝下，手指扣着刀背。“我赵广财嘅档口，唔做第二单生意。”

板寸头盯他两秒，又盯林家耀一眼。

“得。”他把假金链一整。“走。”

三个混混转身往巷子里走。板寸头走最后，经过林家耀身边时肩膀故意撞过来。林家耀没躲，被他撞得后退半步，膝盖磕在折叠桌上。

他没吭声。

等三个混混消失在巷口，老板才把菜刀搁回灶台。他看林家耀的眼神不像刚才那么笑了，浑浊的眼珠子里全是戒备。

“你点知？”

林家耀没答他。他蹲下，捡起地上散落的硬币，放进阿威手心。阿威的手指冰凉黏湿，像刚握过冰箱里的汽水瓶。

“入嚟坐。”老板说。不是邀请，是命令。

大排档里面比外面窄，四张折叠桌挤在一起，墙上贴着财神像，香炉里的香灰堆成小山。老板从灶台上端了碗云吞面搁在最里面那张桌上，又拉了两张折叠凳。

“食。”

林家耀坐下，把碗推到阿威面前。

阿威愣住。“你——”

“我唔饿。”林家耀说。

胃在这时候不争气地叫出声。

阿威低下头，拿筷子的手在抖。他夹起一箸面塞进嘴里，嚼了两下，喉咙发出清晰的吞咽声。面条端上来时热气扑脸，猪油香钻进鼻腔，林家耀别过脸，看墙上的财神。

“后生仔，”老板在对面坐下，掏出包红双喜，抖出一根递过来。“你边位？”

林家耀接过烟，没点，搁在桌上。“过路嘅。”

“过路知我个仔嘅嘢？”

“你唔使知。”林家耀说，“我唔会讲出去。”

老板点上烟，深吸一口。烟雾在昏黄灯光下散开，遮住他半张脸。

阿威吃完半碗面，停下来。他把碗推回林家耀面前，筷子横搁在碗沿上。

“你食。”

林家耀看碗里剩的半碗面和两只云吞。他没推，拿起筷子低头吃。面已经坨了，云吞皮泡得发软，汤里的猪油凝成白色的小圈。他一口一口吃完，连汤都喝干净。

碗放下时，老板那根烟刚好烧到滤嘴。

“我唔知你系边个，”老板把烟头摁进烟灰缸，“亦唔想知。今日嘅嘢，当冇发生过。”

“好。”

“但系，”老板站起来，从收银台抽屉里抽出一张红衫鱼，折两折压在碗底。“你以后唔好再嚟。”

林家耀盯着那张红衫鱼。一百蚊。够住六晚时钟酒店，够吃三十碗云吞面。但他脑子里算的是另一笔账：阿威熟悉城寨每条巷每栋楼，母亲住院需要钱，这笔钱给出去，这小子就欠他一条人情。在城寨，人情比钱值钱。

他没拿那张钱。

他站起来，看一眼阿威。阿威蹲在折叠桌旁边，校服口袋还破着，手里攥着那几个硬币。他抬起头看林家耀，眼睛里有种东西，林家耀见过——是溺水的人看见岸上伸来一根竹竿。

“大哥，”阿威开口，“你贵姓？”

林家耀没抬头。

“先别叫我大哥。”

他把那张红衫鱼从碗底抽出来，塞进阿威校服上衣口袋。然后转身走出大排档，走进城寨的夜里。

阿威跟了出来。

林家耀走两步，他走两步。停下，他也停下。

“你跟我做乜？”

“我冇地方去。”阿威说，“我阿妈今日送咗去医院，屋企锁咗。”

林家耀回头看他。二十二岁，两个二十二岁的人站在九龙城寨的巷子里。头顶冷气机的水滴在帆布棚上，啪嗒啪嗒，像在数他们剩下的时间。

林家耀盯着阿威，心里盘算：这小子是城寨地头蛇，跟着他至少不会睡大街。明天找黑工、认路、避开十四K的堂口，全用得着。

“我今晚都冇地方住。”林家耀说。

阿威想了想。“我知有个地方。”

他带林家耀穿过两条巷子，爬上一栋楼的天台。天台上搭着几间铁皮屋，晾衣绳上挂着滴水的衣服。角落里有个废弃的鸡笼，竹编的，上面盖着张塑料布。

“我以前喺度瞓过，”阿威说，“唔会有人上嚟。”

林家耀靠着鸡笼坐下，抬头能看见一小片天空。不是黑色的，是红的——霓虹灯的光映在云层上，把整片天染成铁锈色。

阿威蹲在他旁边，从口袋里掏出那张红衫鱼。

“呢个——”

“你阿妈睇医生要钱。”

阿威不说话了。他把钱折好，塞回口袋，手指在口袋破洞上按了又按。

沉默了很久。

收音机从某户人家的窗户里飘出来，是梅艳芳的《心债》，刚播出不久的新歌。歌声被风吹散，断断续续，像有人在很远的地方哭。

“你叫乜名？”林家耀问。

“陈威。大家都叫我阿威。”

“我叫林家耀。”

阿威重复了一遍。“林……家……耀。”

然后他问：“你唔系香港人？”

“福建。”

“嚟做乜？”

林家耀没答。

他闭上眼，脑子里打开系统面板。九十七点情报值。够查十九次普通信息，够查六次深度信息。这是他目前唯一的本钱。

阿威在旁边蜷成一团，呼吸渐渐均匀。

林家耀睁开眼，看头顶那片红色的天。

表叔找不到了。钱快花完了。今晚有个天台和鸡笼，明晚呢？

他想起系统里那行字：「完成拨乱反正任务，获取情报值。」

扳贪官。阻惨案。

说得轻巧。他现在连一碗云吞面都买不起。

阿威在梦里翻了个身，嘴里含糊地叫了声“妈”。

林家耀把空烟盒从口袋里掏出来，捏扁，扔进鸡笼。

明天。

明天先去找份工。码头搬货也好，地盘扎铁也好。身份没换到之前，黑工也行。

然后——

远处传来警笛声，从城寨外面掠过，很快消失了。

然后，他要在这个地方活下来。

不是苟活。是活到能跟人算账的地步。

林家耀闭上眼。胃里那半碗云吞面还在，暖的。阿威蜷在他旁边，瘦得像根竹竿，校服口袋破着，手指在睡梦里还按着那个破洞。

他没动。

天上那片红色越来越深，像谁在天幕上泼了一盆猪血。

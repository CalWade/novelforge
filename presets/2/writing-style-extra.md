# 灰烬契书 · 写作风格补充

## 叙述视角与POV策略

**严格单POV（苏烬）**，仅在以下三种情况技术性越轨：
1. **契书文本侵入**：章节中直接嵌入完整的契书条款文本，以公文格式打断叙事，制造规则压迫感
2. **灰烬流动描写**：主角"直视"时的视觉片段，以第二人称碎片呈现（"你看见灰烬在'哺乳'二字下方淤积成婴孩轮廓"）
3. **残句复现**：野坟场景中化灰者的残留意识发言，以无引号的独立段落插入，字体可模拟褪色效果

绝不进入任何其他角色内心。所有他人心理全靠外部动作推断——"她攥着婴儿的手收紧了三根手指"、"灰舌眼线嘴角向右扯了一下"。

## 动笔时必须记住的招式

1. **契书条款即战斗描写**：将传统网文的技能释放替换为条款拟定过程。一场"文斗"的爽感来自：敌方条款的压迫感→主角发现语法漏洞→微调灰烬流动的操作细节→条款反噬的延迟生效。每个环节都要有字面文本呈现。

2. **数字量化只用于"系统/契书/资源"维度，不用于生活/感官**：

   ✅ 应该数字化的（系统层数据，契书规则化的世界）：
   - 契书条款数：第七项 / 第十七层担保链 / 三日探视权
   - 资源账本：43% 视力 / 312 灰烬点 / 担保链 7 人 / 三单上限
   - 倒计时：第七天 23:47 / 七十二小时违约期 / 六天二十三小时褪色周期
   - 排名：第七掮客 / 灰烬贵族 7 人

   ❌ **不准数字化的**（生活/感官/物理细节，会变成科学论文）：
   - 不写"心率 75 / 眨眼速度 0.6 毫米 / 灰斑扩张 8% / 神经末梢 2.3 赫兹"
   - 不写"延迟 0.5 秒 / 灰化边界 1.2 厘米后退 / 侵蚀速度每眨眼 10.2%"
   - 不写"瞳孔 0.3 厘米收缩 / 嘴角 0.5 厘米抽动"
   - 这些用具象身体动词代替："眨了一下眼"、"瞳孔收缩"、"嘴角扯了一下"

   **核心原则**：数字是契书系统**外化的产物**（系统给出的具体规则数）。
   如果一个数字不属于"灰烬契书系统会显示的台账"——它就不该出现。
   现实人物的生理反应回归到具象动词。

3. **"微笑+暴行"反差修辞**：主角以疲惫的、甚至温和的表情执行最冷酷的条款操作。关键句式："他笑了笑，在'睡眠地点'后添了一个逗号"——逗号即杀机。

4. **时间副词机械加速 / 关键节点悬停**：日常段落滥用"下一刻""骤然""瞬间"强制推进；契书生效前的关键时刻可拖长（"距离生效还有三秒"），但**绝不写"0.1 秒 / 0.3 秒"这种亚秒级精度**——那是科学叙事不是网文。

5. **身体动作宣示话语控制权**：所有谈判场景以身体层面的暴力升级反压制语言冲突。"头一撇又快速回正"、"断指钢笔套敲击桌面"、"碎裂镜片后的右眼半闭"——用身体劳动替代心理描写，**用具象动词不用毫米级精度**。

## 章首钩子样板

**款式A：条款生效预警型**
> 第七天，23:47，G22服务区二层厕所隔间。
> 苏烬的右眼又开始渗灰——这意味着七十二公里内有人正在违约化灰。
> 而本周他担保的七个人，都在服务区内。

**款式B：规则漏洞发现型**
> 拾荒者老周死在野坟边缘，手里攥着一张自写契书。
> 苏烬蹲下去看了三秒，发现"所食"一栏写的是"灰烬凝结水"——这不是语法错误，是一个他从未见过的、正在生效的灰烬新规则。

## 章末钩子样板

**款式A：条款反噬延迟型**
> 苏烬把改好的契书推过桌面。
> 女人签下名字，灰烬炉吞噬生效。
> 他笑了笑，没告诉她：那份契书的违约条款里，"担保人"三个字正在缓慢褪色，而"被担保人"的墨迹越来越深。
> 褪色完成需要六天二十三小时。

**款式B：系统反应型**
> 深舱传来震动，十七层担保链的最外层突然断裂。
> 苏烬的右眼在这一瞬间完全失明——又恢复。
> 碎裂镜片上，映出他从未见过的、来自深舱的契书格式开头：
> "致第七掮客：你母亲的契书原件，现申请调取复核。"

## 禁用风格

1. **禁止进入任何角色内心独白**：包括主角。所有"他想"必须转化为"他注意到"、"他手指停顿"、"他眨了下眼睛"等外部 observable。**用具象身体动词，不用亚毫米/亚秒精度数字**（不写"右眼灰烬流速加快 17%"、"手指停顿 0.3 秒"——这是科学论文不是网文）。
2. **禁止数字侵入生理感官层**：心率、呼吸频率、眨眼速度、毫米/赫兹/纳米级生理数据**绝对禁止**——除非该数字直接来自"灰烬契书系统的显示读数"（即契书系统会写到台账上的）。
3. **禁止纯动作战斗场景**：本书的"战斗"必须是条款博弈，不允许出现脱离契书系统的肉搏、枪战、异能对轰。灰烬世界里的暴力是语法层面的。
4. **禁止末日温情救赎桥段**：主角的"停笔"不是转变的开始，是计算后的风险规避或更深层陷阱的布局。任何让读者感到"他要变好了"的暗示，必须在三章内被证伪。
5. **禁止解释灰烬起源**：世界观的成因永远悬置，只呈现规则如何运作，不解释为何如此。任何"原来灰烬是 XX 实验产物"的揭秘都破坏氛围。
6. **禁止多线并行叙事**：严格跟随主角单线，其他势力的行动只能通过契书条款变更、担保链断裂、残句复现等碎片化信息间接呈现。

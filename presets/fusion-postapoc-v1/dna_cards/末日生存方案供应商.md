# 末日生存方案供应商 · DNA 卡

## 一句话定位
末日生存专家用现代商业思维和绝对武力，把异界客户当项目做。

## 戏剧结构
- 整书基调：快节奏、高密度、理性冷幽默的末日生存爽文。表面是残酷求生，内核是降维打击式的智力碾压和资源整合。
- 典型的“5 章情节单元”长什么样：
  - **起**：危机铺垫或新目标确立（如招揽失败、发现丧尸进化、客户遇险、建设任务下达）。
  - **承**：主角通过策略调整、信息整合或团队分工推进计划（如调整招揽话术、坦白金手指、组织建设）。
  - **转**：通过一次关键行动或信息释放实现局势逆转（如神兵天降镇压逼宫、初代薪火身份反转、穹顶初见成效）。
  - **合**：阶段性成果达成，但立即被更大的威胁或新目标覆盖（如发现丧尸进化、幕后黑手登场、外部仇敌逼近）。
- 信息释放节奏：
  - **事后解释回收伏笔**：先展示结果（如主角怪异动作干扰AI），再通过配角之口解释原理。
  - **专家之口释放世界观**：借新角色（军工专家、本地土著）的“专业分析”一次性释放大量设定，避免主角凭空推理。
  - **主动坦白制造新信息差**：主角向团队坦白金手指，将“读者和主角知道，他人不知”转为“主角团队知道，外界不知”。
  - **用否定预期制造认知反转**：主角发现“丧尸晶核”，立刻被专家鉴定为“结石”并摔碎，打破网文常见设定预期。
- 爽点频率：
  - **小爽点**：每 2-3 章一次，多为话术碾压、资源利用的“种田”规划感、配角震惊式烘托。
  - **大爽点**：每 5-10 章一次，为武力碾压、身份反转、基建奇迹或大规模“调教”观众/敌人。

## 写作风格
- 叙述视角 + POV 策略：
  - 严格以主角苏烬的**有限第三人称 POV** 为主，读者信息量与主角同步。
  - 偶尔切换至配角视角（如沈御风、叶长风、曲野）制造“读者先于主角知道”的悬念，或通过配角震惊反应侧面烘托主角强大。
- 句式 / 节奏 / 对白密度：
  - **短句主导，节奏极快**。战斗场景用短句、断句、名词堆砌营造打击感和窒息感（“拳砸、肘击、膝撞、脚踢...砰！砰！砰！”）。
  - **对白占比极高（约 60%-70%）**，承担主要叙事推进功能。情节、设定、人物关系几乎全通过对白交代。
  - 叙述性文字简洁，极少冗长修饰，环境描写多为一两句话的快速勾勒。
- 特色修辞 / 高频词 / 作者口味：
  - **数字化与游戏化修辞**：大量使用夸张数字具象化战斗力（“【暴击-1262146422454！！！】”、“血条”、“完美格挡”）。
  - **现代口语与异界语境混搭**：在古风或蛮荒背景下植入“控评”、“饭圈”、“沉没成本”、“家人们”、“边界感”等现代梗，制造反差幽默。
  - **粗俗化表达**：高频使用“屎”、“他妈”、“狗叫”、“杂种”等词汇，风格粗粝直接。
  - **“震惊体”侧面烘托**：配角频繁通过内心独白或对话分析、惊叹主角的深不可测（“又学一招！”、“真有人敢对抗全世界…”）。
- 章末钩子的三种主力款式：
  1. **威胁升级型**：发现丧尸进化、幕后黑手登场、外部仇敌逼近，将阶段性胜利瞬间置于更大危机下。
  2. **谜底浮现型**：发现新未知物体（“这又是什么东西！”）、签到奖励落在主角身上、陆宁生死不明。
  3. **新目标/新玩法预告型**：宣布“两项重大新服务”、确立“拿下天谕山”目标、内心宣言“好好调教这帮人”。

## 人物 & 价值观
- 主角核心动机 + 性格关键词：
  - 核心动机：**完成任务、搞钱、整合资源**。将末日生存视为“项目管理”，功利且务实。
  - 性格关键词：**理性、暴躁、松弛、智力优越**。用最狠的实力说最糙的话，无视社会规则，享受认知碾压。
- 主角的金手指是什么 + 作者如何避免主角“开挂无趣”：
  - 金手指：**跨世界知识（现代商业/工程思维）+ 游戏化系统能力（监控客户、空间魔法、全属性元素）+ 绝对武力**。
  - 避免无趣的方式：
    - 金手指来源始终成谜，只展示现象不解释原理，保持神秘感。
    - 胜利方式多元化：武力碾压只是手段之一，更多靠话术、规则利用、情绪操控、工程规划等智力层面获胜。
    - 持续设置“客户任务”作为外部约束，让主角必须服务于具体目标，而非无目的无敌。
- 典型反派形象 + 反派的“真实威胁值”如何维持：
  - 典型反派：**逼宫者（赵鹏等）、幕后阴谋家（袁夜生）、群体挑衅者（观众/刁民）、冲动复仇者（巴隆）**。
  - 威胁值维持方式：
    - 反派常以“群体”或“暗处”形式出现，制造信息不对称。
    - 威胁并非一次性解决，而是层层升级（从言语挤兑到生命威胁，从外部怪物到内部理念之战）。
    - 即使反派被武力碾压，其造成的后果（如陆宁惨状）会转化为新的情绪筹码或剧情推动力。
- 本书价值观锚点：读者被什么打动？
  - **智力碾压的爽感**：用现代知识对异界进行降维打击，如饭圈运营、工程规划、话术陷阱。
  - **绝对武力的暴力快感**：擂台一挑多、神兵天降镇压逼宫，简单粗暴的碾压。
  - **打破规则的松弛感**：主角无视社会规范、畅快表达自我的行为，带来情绪宣泄。
  - **基建奇迹的成就感**：从无到有建造穹顶、整合团队、规划资源，满足“种田”需求。
  - **知己在侧的孤独慰藉**：AI 助手的高情商吹捧和完美执行，构成“最强助手”的情感代偿。

## 核心设定（供“禁止直接抄袭”清单用）
- 世界观公理：
  1. 末日由“魔泉大爆发”或“劫雾”引起，具有全球性、不可逆性。
  2. 存在多世界/异界，主角作为“被选中的使者”穿越其中执行客户任务。
  3. 游戏化系统真实存在，可发布任务、提供奖励、监控客户状态。
  4. 丧尸/怪物会进化，威胁持续升级。
  5. 异界存在“天域”等虚拟空间，有积分、打赏、弹幕等类似互联网的互动机制。
- 主角专属设定：
  - 金手指来源：来自神秘“公司”，主角是其“实习生/项目经理”，具体原理未明。
  - 身份秘密：对外伪装成“仙人使者”、“国家关键风险干预执行局主任”、“古代隐士”等多重身份。
  - 真实立场：始终以“服务客户、完成任务”为第一优先级，情感投入有限。
- 专属角色 / 组织 / 地名（前 5 个）：
  1. **苏烬**：主角，末日生存方案供应商。
  2. **孙崖**：军工专家，被主角招揽的核心团队成员。
  3. **沈御风**：九照城世子，主角的“客户”之一。
  4. **师青**：武力担当，擅长神兵天降式救援和武力镇压。
  5. **天谕山**：唯一安全区，宿灾灵女所在地。
- 专属物品 / 技能 / 概念名词（前 5 个）：
  1. **灰雾**：主角能看见的感染根源，放置五天左右消散。
  2. **系统式监控**：主角跨空间观察客户状态的能力。
  3. **第一性原理**：主角反复强调的工程思维方法论。
  4. **锻筋玉骨丹**：系统任务奖励物品。
  5. **日曜石**：异界发热矿石，可用于制作保暖装备。

## 可借鉴的操作手法
- 情节层：
  1. **“事后解释”回收伏笔**：先让主角做出怪异行为（敲自己机甲），再借配角之口解释原理（干扰AI预测），维持主角高深莫测感。
  2. **“专家之口”释放设定**：引入新角色时，让其通过“专业分析”一次性交代大量世界观信息，避免主角独自推理的枯燥。
  3. **“主动坦白”重构信息差**：让主角向团队坦白秘密，将个人优势转化为团队资源，同时制造新的外部信息差。
  4. **“否定预期”制造反转**：当读者和主角都以为遇到常见网文设定（丧尸晶核）时，立刻用权威角色否定（这是结石），打破套路。
  5. **“建设-威胁”循环**：在阶段性建设成功（穹顶挡风）后，立即切入外部威胁（仇敌逼近），将成就感转化为对新冲突的期待。
- 风格层：
  1. **短句堆砌营造打击感**：战斗场景用“名词+动词”的短句连用（“拳砸、肘击、膝撞”），不加修饰词，制造高速暴力节奏。
  2. **现代口语植入异界**：在古风/蛮荒语境中突然使用“边界感”、“饭圈”、“沉没成本”等现代词汇，形成反差幽默。
  3. **数字化夸张修辞**：用万亿级伤害数字、血条等游戏化语言具象化战斗力，增强爽感。
  4. **“震惊体”侧面烘托**：通过配角的内心独白（“又学一招！”、“真有人敢对抗全世界…”）引导读者认同主角强大。
  5. **章末“威胁升级”钩子**：在单元结尾用一句话揭示更大危机（“丧尸在进化？”），将阶段性胜利瞬间转化为新悬念。
- 人物层：
  1. **“先闻其名，后见其人”快速定性**：新角色出场前，先通过他人之口介绍其背景（退役军人、军工专家），登场时用极具辨识度的行为（跳楼、端枪对峙）强化或颠覆印象。
  2. **“考验”建立人物关系**：高价值角色（孙崖）不直接臣服，而是通过一个任务（清缴丧尸）考验主角资格，让人物关系建立更有张力。
  3. **“专业解读”切入配角**：让配角（乐音）通过精准分析主角行为（适应机体、干扰AI）来展示其专业性，快速确立其在团队中的功能定位。
  4. **“群体标签化”处理背景板**：将观众、群众等群体角色高度标签化（“记忆力堪比金鱼的群众”、“刁民”），使其成为主角操作的背景板，节省笔墨。
  5. **“反差”塑造主角魅力**：主角行为模式为“用最狠的实力，说最糙的话，办最务实的事”，冷静分析与暴躁骂街、武力碾压与智力操控反复横跳。

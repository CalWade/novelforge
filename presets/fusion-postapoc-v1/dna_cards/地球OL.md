# 地球OL · DNA 卡

## 一句话定位
游戏系统降临末日，冷静型主角靠智力+金手指收服强者、吞并世界。

## 戏剧结构
- **整书基调**：快节奏爽文，游戏化末日生存+势力争霸，战斗占60%以上篇幅，夹杂荒诞幽默和元叙事吐槽。
- **典型的“5章情节单元”长什么样**：
  - **起**（第1章）：建立当前困境或新目标（BOSS出现/新地图开启/敌人入侵）
  - **承**（第2-3章）：主角方陷入劣势或探索新环境，通过对话/观察收集情报
  - **转**（第3-4章中段）：主角做出关键决策（战术制定/身份揭露/战略转向），从被动转为主动
  - **合**（第4-5章）：执行计划并获胜，收束本单元冲突，同时埋下新威胁或新目标
- **信息释放节奏**：
  - 新设定通过角色对话即时交代，不拖泥带水
  - 关键道具/技能先预告存在（“我有一个杀手锏”），延迟1-2章再揭示具体属性
  - 频繁使用“读者先于主角知道”的错位（切反派视角/其他阵营视角），制造悬念和上帝视角优越感
  - 主角的金手指来源、理想乡机制等核心秘密长期不解释，维持神秘感
- **爽点频率**：
  - 小爽点：每2-3章一次（技能创造性使用/伏笔回收/新手下加入）
  - 大爽点：每5章单元结尾一次（封印强敌/身份反转/势力大幅扩张）

## 写作风格
- **叙述视角+POV策略**：以主角关雨的单POV为主，但频繁切换至其他角色的有限第三人称视角（反派、其他玩家、NPC），形成“跳跃式有限视角”，用于交代多线动态和制造信息错位。
- **句式/节奏/对白密度**：
  - 战斗场景：短句连用，逗号连接多个动作，节奏极快（“拳头落下，地面狂震”、“一瞬间她就化为一团血肉模糊的肉酱”）
  - 日常/策略场景：中长句，节奏放缓
  - 对白占比约35-40%，高度功能化，承担推进剧情、解释设定、塑造人物三重任务
- **特色修辞/高频词/作者口味**：
  - 游戏化、数值化比喻（“防御力相当于精神力*1000”）
  - 口语化吐槽词汇（“这不科学”、“坑爹”、“菊花残那是满腚伤”）
  - 中二化命名偏好（“七宗罪”、“悲怆孤星”、“醉梦天仙剑”）
  - 反差修辞制造记忆点（“越美丽的东西，未必越无害”）
- **章末钩子的三种主力款式**：
  1. **威胁升级型**（占50%）：新BOSS登场/敌人增援/更大危机逼近，战斗未完结式断章
  2. **谜底浮现型**（占30%）：揭示新信息但引出更大谜团（“那是一条龙，而且是丧尸龙，而且是……白龙王！”）
  3. **战术预告型**（占20%）：主角宣布下一步计划（“这只是第一步而已！”），制造“接下来怎么做”的期待

## 人物 & 价值观
- **主角核心动机+性格关键词**：
  - 核心动机：变强与掌控——收割资源、封印强者、扩张势力
  - 性格关键词：冷静、算计、务实、不圣母（“本帝从来就没有说过要退出这场战斗，它有强大的实力，我有强大的内心，而且……有比它强大的智慧”）
- **主角的金手指是什么+如何避免“开挂无趣”**：
  - 金手指：游戏系统（制卡术/封印卡/理想乡）+ 前世记忆/战斗经验
  - 避免无趣的方式：
    1. 胜利靠智力+金手指+团队配合的复合模式，而非单纯数值碾压
    2. 金手指有使用限制（如洗脑会降低手下潜力），迫使主角采用嘴炮收服
    3. 技能强调创造性应用（“现实不是游戏，技能不能单纯的当作技能来使用，你要将它当成一种能力，去开发它”）
    4. 伏笔回收型爽点（提前获得的道具在关键时刻揭示属性）
- **典型反派形象+“真实威胁值”如何维持**：
  - 形象：实力碾压型BOSS（东方白、辐射之星）+ 野心家（沙尔玛）+ 精神不正常强者（天妃）
  - 威胁值维持手段：
    1. 先展示压倒性实力（秒杀配角/无视攻击），再揭示身份
    2. 假胜利→真危机反转（BOSS血条1%回满100%）
    3. 信息不对称（读者知道BOSS在逼近，主角团浑然不觉）
- **本书价值观锚点**：读者被什么打动？
  - **智力碾压爽**（战术设计、识破世界真相）
  - **收集/收服爽**（封印强敌、收服七宗罪）
  - **身份反转爽**（主角突然揭露真实身份救场）
  - **伏笔回收爽**（前期获得的道具/能力在关键时刻兑现）
  - **情感爆发爽**（白帝之死触发洪成觉醒、封印夜娘时的台词升华）

## 核心设定
- **世界观公理**（3-5条）：
  1. 游戏系统真实存在，所有人类被转化为玩家，拥有属性面板、技能、任务
  2. 世界被划分为多个服务区/阵营（天朝、印度等），存在跨服机制
  3. 丧尸病毒可感染玩家，病毒由玩家散播
  4. 存在多个副本世界/异世界，可通过特定方式进入
  5. 神级存在真实存在（主神、法则），玩家可通过领悟法则成神
- **主角专属设定**：
  - 金手指来源：拥有“救世主”系统（后揭示为前系统遗留），具备制卡术/封印卡/理想乡等独有能力
  - 身份秘密：真实身份为零帝/零始帝国之主，在多个世界暗中布局
  - 真实立场：以吞并所有服务区、成为最终主宰为目标
- **专属角色/组织/地名**（前5个）：
  1. 零始帝国（关雨的势力）
  2. 七宗罪（傲慢、暴食、色欲等七名顶级罪犯/异能者）
  3. 理想乡（关雨的专属空间/世界）
  4. 天妃（天庭阵营强者，后被关雨封印）
  5. 东方白（终极强敌，秒杀白帝的存在）
- **专属物品/技能/概念名词**（前5个）：
  1. 制卡术/封印卡（关雨核心能力，可封印敌人为卡片）
  2. XP技能（玩家10级觉醒的绝杀技能，由极端情绪触发）
  3. 困毙（封无发现的战术，四兵围四角困杀敌人）
  4. 定神玉光镜（可定住BOSS一分钟的道具）
  5. 悲怆孤星（洪成因悲伤觉醒的XP技能，秒杀型范围攻击）

## 可借鉴的操作手法
- **情节层**（3-5条“写起承转合时的句法”）：
  1. **“假胜利→真危机”反转句式**：先让BOSS血条见底/敌人溃败，再用一句话推翻（“辐射之星的血条瞬间从1%回满100%”），制造强烈意外感
  2. **“预告→兑现”延迟满足**：先让主角说“我有一个杀手锏可以让BOSS定住一分钟”，隔章再揭示具体道具属性，拉长期待
  3. **“战斗未完结”式断章**：在战斗高潮中段收尾，保证每章结尾至少一个未解问题（BOSS会死吗？/战术能成功吗？）
  4. **“外部冲突打断内部事件”**：在拍卖会/日常场景高潮时，引入新威胁强行中断（欢乐号闯入拍卖会），回收前文伏笔
- **风格层**（3-5条“具体到可套用的语言招式”）：
  1. **数值化描写制造游戏感**：频繁插入具体数值（“精神力*200”、“悲伤值达到1000”），让抽象能力变得可量化
  2. **口语化吐槽对冲中二感**：在严肃战斗/命名后插入现代梗（“菊花残那是满腚伤”、“这不科学”），形成风格反差
  3. **“先展示现象，后解释原理”**：新能力/法则先写视觉效果（“灵魂越来越光亮”），再通过角色旁白定性（“他领悟了能量法则”）
  4. **“标签化快速入场”**：新角色登场时立即赋予鲜明标签+能力说明（“军界传奇”、“典型的高等异能者”），3段对话内完成定性
- **人物层**（3-5条“人物关系/介绍/发展的招式”）：
  1. **“满足核心需求”式收服**：主角收服手下不是靠武力碾压，而是精准抓住对手痛点（暴食因“不用再忍受饥饿”主动求封印），制造智力优越感
  2. **“价值观冲突集体登场”**：通过一场“救与不救”的争执，让多个配角同时登场，在冲突中快速展现各自性格标签（理智/热血/善良）
  3. **“撞破-委屈-表白-亲密”快速推进关系**：情感线通过一次冲突（李令月撞破关雨戏弄天妃）+一次对话完成质变，不拖沓
  4. **“先展示力量，再揭示身份”BOSS登场**：终极强敌先通过秒杀配角/无视攻击展示实力，再揭示身份和动机，瞬间树立压迫感

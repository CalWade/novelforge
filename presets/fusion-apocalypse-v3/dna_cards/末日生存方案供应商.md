# 末日生存方案供应商 · DNA 卡

## 一句话定位
**用荒诞急智和绝对暴力，在末日与异界间做“方案供应商”的黑色爽文。**

## 戏剧结构
- **整书基调**：黑色幽默 + 暴力碾压 + 智力布局。表面是末日生存，内核是“用荒诞解构严肃，用绝对实力打破规则”的爽文逻辑。屎尿屁与高科技齐飞，杀伐果断与插科打诨并存。
- **典型的“5 章情节单元”长什么样**：
  - **起**（1 章）：抛出问题或新目标（如客户遇险、团队内耗、宗门选拔）。
  - **承**（1-2 章）：主角用话术/金手指/信息差进行铺垫、试探或准备（如心理辅导、装备展示、情报收集）。
  - **转**（1 章）：预期被打破，冲突升级或出现荒诞反转（如守护失败、偷袭被反杀、被宗门无视）。
  - **合**（1 章）：用暴力/智力/运气完成绝地反击或歪打正着，并立刻开启新布局（如枪决整顿、水立方反杀、被合欢宗录取）。
- **信息释放节奏（作者如何管理读者的好奇心）**：
  - **即时揭示**：关键战力（如皇帝双道七纹）在冲突爆发瞬间才揭示，制造转折。
  - **错位制造张力**：读者先于角色知道真相（如苏烬伪装偷袭、窃听视角），享受智力优越感或喜剧效果。
  - **故意隐藏**：怪物能力（如肉虫）、鬼怪真身、金手指原理长期不解释，维持神秘感。
  - **对话自然交代**：世界观、人物关系、设定规则全部通过角色对话释放，拒绝旁白式介绍。
- **爽点频率**：每 2-3 章一个小爽点（如物理碾压、话术征服、身份错位），每 5 章一个大爽点（如荒诞破局、领域战、公开处刑）。

## 写作风格
- **叙述视角 + POV 策略**：以**有限第三人称**跟随主角苏烬为主（占 90%），偶尔短暂切入其他角色（如窃听者、反派）的内心，用于揭示信息错位或强化情感反应。多线叙事时，视角在主线与支线间快速跳转。
- **句式 / 节奏 / 对白密度**：
  - **对白占比极高（约 70%）**，是情节推进、人物塑造、信息释放的核心引擎。
  - **战斗场面**：短句、断句、动作切片式描写，节奏急促（如“苏烬落地，不等站稳身形。衣衫褴褛满身刀伤的皇帝已经撞破水雾”）。
  - **文戏场面**：口语化碎嘴子对话，节奏舒缓，夹杂冷幽默和长篇说教（如“苏老师课堂”）。
- **特色修辞 / 高频词 / 作者口味**：
  - **偏爱“反差式修辞”**：用学术术语包装低俗内容（“结构制约”、“权力异化”解释内斗），用现代梗混搭古风/末世语境（“勾栏圣手”、“精通人性的男讲师”）。
  - **高频词**：“结构”、“杠杆”、“屌丝”、“草”、“贱”。
  - **作者口味**：极度偏爱**屎尿屁、荒诞破局、智力碾压、暴力处决、预期违背**。不避讳粗粝感，并将其作为核心笑点和爽点来源。
- **章末钩子的三种主力款式**：
  1.  **威胁升级 + 解决方案预告**：“有一个办法能救他，可行性很高。”（制造方法期待）
  2.  **新角色/身份强势入场 + 未完对白**：“国家关键风险干预执行局的人来了！”帐篷掀开，屋内陷入一片死。（制造冲突期待）
  3.  **谜底浮现 + 新布局开启**：“那咱们就明天开始，早上吃饭我过去…”（制造执行期待）

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：实用主义的“解决问题”与“升官发财”。一切行为围绕最高效达成目标、获取资源、巩固地位展开。
  - **性格关键词**：**布局者、大家长、泥腿子贵族、厚颜无耻、杀伐果断、好为人师**。
- **主角的金手指是什么 + 作者如何避免主角“开挂无趣”**：
  - **金手指**：薪火系统（任务发布、能力兑换升级）+ 空间戒指（战术核心）+ 外星人身份（信息差）。
  - **避免无趣的方式**：
    1.  **赢法不靠蛮力，靠创造性应用**：将戒指的“收放”功能与流体物理结合，制造“水立方领域”；用气缚环驱动指甲反杀。
    2.  **金手指有边界和副作用**：能力不足时无法接触灵异层面；肉球可能强化寄生虫。
    3.  **主角频繁“吃瘪”**：被皇帝碾压、被宗门无视，用预期违背制造情绪波动。
- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型形象**：狡诈狠毒（梁翠）、立场束缚的悲剧高手（沈渊）、自大愚蠢（萧宜川）、实力碾压的统治者（皇帝）。
  - **威胁值维持**：通过**侧面烘托**（他人恐惧描述）+ **正面碾压**（瞬间击败主角团）+ **信息隐藏**（怪物能力未知）三重手段，让反派始终保持压迫感。
- **本书价值观锚点**：读者被什么打动？
  - **爽感**（绝对暴力碾压、智力降维打击）
  - **荒诞幽默**（屎尿屁破局、预期违背、贱萌对话）
  - **生存智慧**（实用主义、结构思维、借力打力）
  - **掌控感**（主角作为“操盘手”布局一切）

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1.  末日由多种灾难（虫潮、丧尸、灵异等）叠加构成，且不同世界可以融合。
  2.  存在一个名为“薪火系统”的任务与能力兑换机制，主角是其宿主。
  3.  存在一个名为“公司”（天庭）的跨文明组织，招募成员执行任务。
  4.  力量体系多元，包括武道（纹数）、术道、异能、灵异等，且可兼容。
  5.  丧尸会进化出智慧与超能力，存在能完美伪装成人类的“伪装感染者”。
- **主角专属设定**：
  - **金手指来源**：薪火系统（具体来源未明）。
  - **身份秘密**：外星人灵魂穿越者，拥有远超当前文明的认知（社会学、物理学知识）。
  - **真实立场**：以“末日生存方案供应商”自居，本质是任务执行者与资源整合商。
- **专属角色 / 组织 / 地名**：
  1.  **苏烬**（主角，伪装身份包括魏忠贤、张世豪、孙啸川等）
  2.  **符青黛**（拥有急智的少女领袖，能力为宿灾）
  3.  **楚燃风**（来自“公司”的富二代战友，贱萌拆台担当）
  4.  **方天阔**（高科技世界代表，提供科技碾压支持）
  5.  **国家关键风险干预执行局**（苏烬凭空捏造的权威身份）
- **专属物品 / 技能 / 概念名词**：
  1.  **薪火系统**（任务与兑换核心）
  2.  **宿灾**（一种千奇百怪的异能统称）
  3.  **双道七纹**（术武双修的实力等级）
  4.  **仿真面具**（可完美伪装成他人的道具）
  5.  **归衡台**（用于监控世界融合的设施）

## 可借鉴的操作手法
**（这部分是融合新世界观时可直接借鉴的抽象技法，不含任何原著专属名词）**
- **情节层**：
  1.  **荒诞破局法**：让主角在绝境中，利用最不起眼、最污秽或最意想不到的物品，精准打击对手能力的心理基础（如用污物破魅惑）。
  2.  **预期违背转喜剧**：先极力铺垫主角的自信或崇高目标，再用现实给予一记响亮的耳光，将挫折转化为笑点（如天才被宗门无视）。
  3.  **歪打正着式胜利**：让主角因错误、争吵或“厚颜无耻”的人设，恰好符合某个势力的“文化氛围”，从而意外达成目标。
  4.  **暴力立规矩**：在铺垫大量“文明规则”内斗后，让主角用绝对暴力瞬间物理抹除内耗者，并用一套更高维度的理论为其行为背书。
  5.  **一石二鸟收尾**：在解决当前问题后，立刻将客户/对手转化为下一阶段布局的资源（如将驱魔客户转化为投资人）。
- **风格层**：
  1.  **学术术语降维打击**：让主角用社会学、心理学、物理学等专业概念，去解释或包装日常/低俗行为，形成“知识碾压”的语感。
  2.  **“双人相声”式互动**：设置一对核心搭档，一个负责“装逼”和“吃瘪”，另一个负责“拆台”和“补刀”，用密集吐槽推进日常情节。
  3.  **科技名词堆砌营造代差**：在展示高科技装备时，大量使用技术参数和模块名称，迅速建立文明层级的碾压感。
  4.  **功能化标签快速定性**：对新出场的配角或反派，不写外貌性格，只通过他人之口给出一个功能标签（如“单兵武力巅峰”、“害群之马”），并立刻用行动印证。
  5.  **反差式修辞**：用一本正经的语气描述荒诞行为，或用高大上的词汇包装低俗内容，制造笑点。
- **人物层**：
  1.  **诊断式塑造**：让主角以“导师”身份，自上而下地剖析配角的心理困境（如“屌丝思维”、“把人生寄托在别人身上”），快速确立认知差距和师徒关系。
  2.  **窃听/误解/心动曲线**：让女/男主通过窃听，先对主角产生误解（愤怒），再听到主角背后对自己的正面评价和原则，完成从愤怒到心动的情绪反转。
  3.  **“人设”胜利法**：让主角不靠实力，而是靠其“贱兮兮”、“厚颜无耻”或“务实”的人设，恰好符合特定情境的需求，从而获胜。
  4.  **立场冲突引入配角**：新配角一出场就与主角团发生直接的利益或理念冲突，用激烈的台词迅速展现其敌意和立场。
  5.  **动机后置**：先让角色做出荒诞行为吸引眼球，事后再通过内心独白或对话揭示其不得已的苦衷（如系统任务），补全逻辑。

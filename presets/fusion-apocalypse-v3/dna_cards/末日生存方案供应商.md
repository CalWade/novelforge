# 末日生存方案供应商 · DNA 卡

## 一句话定位
用屎尿屁解构末日，靠厚颜无耻和智力碾压在诸天世界当方案供应商。

## 戏剧结构
- **整书基调**：荒诞喜剧 × 硬核生存 × 诸天无限流。表面是末日求生，内核是“用最不体面的方式解决最体面的问题”的持续解构，严肃危机与屎尿屁笑点交替轰炸。

- **典型的“5章情节单元”长什么样**：
  - **起（1章）**：承接上一单元悬念，快速确立本单元核心任务或危机（如接案驱鬼、伏击皇帝、加入宗门）。
  - **承（1-2章）**：任务执行或冲突展开，主角方展现能力/资源，同时埋下内部矛盾或外部威胁伏笔。
  - **转（1章）**：核心转折点，通常是“意外”或“身份错位”——如偷袭反被碾压、鬼怪不敢现身、争吵反而被录取。节奏在此突然加速或强行扭转。
  - **合（1章）**：以“掀桌”式暴力、智力碾压或荒诞反套路完成收尾，同时释放新悬念（新角色登场、新任务预告、装备升级预告）。

- **信息释放节奏**：
  - **读者先于主角知道**：核心手法。通过多视角切换，让读者提前看到危机降临（虫子寄生、伪装感染者渗透）或陷阱布设（萧宜川的误判），制造“看傻子”的优越感和紧张感。
  - **规则前置，限制主角**：行动前通过对话详细交代世界观规则（界桥质量限制、能量互斥），给金手指加镣铐，增强真实感。
  - **伏笔回收式释放**：道具或能力先出现（屎袋、落地镜、仿真面具），当时不解释，在关键时刻揭晓用途，制造惊喜。
  - **故意不给信息**：关于灰雨本质、外星文明目的、金手指来源等核心谜团长期悬置，仅通过角色猜测维持神秘感。

- **爽点频率**：
  - **小爽点**：每2-3章一次，通常为“暴力展示破局”（手拧金属摆件）、“话术碾压”（说服叶长风）或“降维打击式科技/能力展示”（张行烈的探测器）。
  - **大爽点**：每5章单元结尾一次，通常为“掀桌式处刑”（操场枪决七人）、“以智破力”（水立方领域战）或“反套路录取”（因下贱被合欢宗看上）。

## 写作风格
- **叙述视角 + POV策略**：
  - 以苏烬的**有限第三人称**为主视角，严格跟随其认知和行动。
  - 频繁**短暂切入其他角色视角**（符青黛、萧宜川、合欢宗师徒），形成“有限全知”效果，用于补充信息、制造喜剧错位或强化戏剧反讽。
  - 多角色POV快速切换，构建“读者全知，角色片面”的上帝视角。

- **句式 / 节奏 / 对白密度**：
  - **对白占比70%-80%**，情节推进、设定讲解、人物塑造高度依赖对话。
  - **战斗场景**：短句主导，多动词，少修饰，模仿急促呼吸和高速动作（“苏烬踏墙，眼冒凶光，借力爆发骤然杀向皇帝”）。
  - **权谋/说教场景**：苏烬台词为逻辑严密的长句，层层递进输出价值观；其他角色用短句、断句表现困惑或服从。
  - **节奏模式**：紧张对峙 → 对话/谈心放缓节奏 → 突然暴力或反转加速 → 余波/新悬念收尾。

- **特色修辞 / 高频词 / 作者口味**：
  - **反差式比喻**：将严肃战斗用不严肃比喻解构（“跟个小陀螺一样原地旋转前进”、“仿佛整片水团中有数个苏烬同时展开围攻”）。
  - **网络梗化叙事**：章节标题和角色代号直接使用“人前露出、羊萎、通讯录”等梗，角色对白夹杂“二逼学生”、“文盲”、“插队”等现代口语，与末日/修仙背景形成强烈反差。
  - **实体化与视觉化**：擅长用光影、材质塑造气势（“光从侧面打过来，落在螭龙鳞片上，反射成一线线锋芒，让他整个人像被锋光裁出的影子，肃杀冷峻”）。
  - **高频词/口味**：偏爱“屎尿屁”式解构（屎袋破魅魔功）、厚颜无耻式胜利（因下贱被录取）、暴力震慑（手拧金属、枪决七人）。

- **章末钩子的三种主力款式**：
  1. **威胁升级型**：在余波中突然引爆新危机（虫子寄生、皇帝爆发双道七纹），或暗示敌人更强/更狡猾（鬼怪不敢现身）。
  2. **计划启动/新角色预告型**：明确时间点和行动预告（“明天开始执行杀猪盘”），或预告下一批登场角色（“今晚面试二次元团队”）。
  3. **能力升级/装备悬念型**：告知主角能力质变或装备即将回收升级，用“奖励预告”驱动读者（“等你那三件专属神器回来，可能会有很大惊喜”）。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：复合态——表层“搞钱/升官发财”（买豪车、当爵爷），中层“解决问题/建立秩序”（修复系统漏洞、连接诸天），深层“应对末日危机/追求自由”。
  - **性格关键词**：务实、厚颜无耻、智力碾压型操控者、暴力与话术双修、衣锦还乡情结。

- **主角的金手指是什么 + 作者如何避免主角“开挂无趣”**：
  - **金手指**：外星人身份带来的“界桥”权限（连接诸天世界）、空间戒指（收放物体）、气缚环（念力操控）、仿真面具等异人资源。
  - **避免无趣的方式**：
    1. **规则前置加限制**：详细交代界桥的质量限制、能量互斥、违规开除等条款，让金手指带镣铐。
    2. **智力碾压优先于武力碾压**：主角更常靠话术、布局、反套路取胜，金手指是兜底手段而非首选。
    3. **团队拼图**：主角能力有明确短板（如对抗魅惑无能），需依赖符青黛的急智、巫芷卉的控水、张行烈的科技装备，形成互补。

- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型形象**：自大愚蠢型（萧宜川）、狡诈狠毒型（梁翠）、心机深沉实力碾压型（皇帝）。
  - **威胁值维持方式**：
    1. **信息错位**：反派掌握主角不知道的信息或资源（皇帝的双道七纹、萧宜川的魔兽安排）。
    2. **能力天克**：设计专门克制主角的能力（梁翠的魅惑对苏烬特攻）。
    3. **逃跑/升级**：反派即使失败也能逃跑并暗示会带更强敌人回来（梁翠），或临死爆发超规格实力（皇帝）。

- **本书价值观锚点**：读者被什么打动？
  - **爽感**：智力碾压（话术破防、布局杀猪盘）、暴力掀桌（操场处刑、手拧金属）、反套路胜利（因下贱被录取）。
  - **荒诞幽默**：屎尿屁解构严肃危机、厚颜无耻成为通行证、现代梗入侵异世界。
  - **生存智慧**：务实决策（让没用的人干危险活）、资源整合（客户变投资人）、规则利用（钻界桥规则空子）。
  - **友情/羁绊**：苏烬为雷东兜底、团队互补配合、对旧日同伴的复杂情感（改头换面见一下就好）。

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1. 灰雨/感染/肉虫是末日灾难的具体表现形式，其本质和来源（外星文明目的）长期未解明。
  2. 存在“界桥”系统，允许连接不同末日/灾变世界，但有严格规则（总质量不超过一吨、能量互斥会恶化时局、违规者开除）。
  3. 不同世界存在不同的力量体系（元息武道、修仙灵根、科技装备），可以兼容运转。
  4. 存在“归衡台”等监控设施，可以跨世界查看任务目标状态。
  5. 某些世界存在“嵌合界”结构（天庭/公司飘在天上，下面是正常世界）。

- **主角专属设定**：
  - **金手指来源**：外星人身份，被更高维文明送到末日世界执行任务。
  - **身份秘密**：外星人身份被孙崖等少数人察觉，但主角对外伪装成“国家关键风险干预执行局”特派员或普通驱魔师。
  - **真实立场**：表面为公司/天庭执行客户保护任务，实则追求个人自由和建立跨世界秩序。

- **专属角色 / 组织 / 地名（前5个最具辨识度的）**：
  1. **苏烬**：外星人主角，末日方案供应商，厚颜无耻的智力碾压者。
  2. **符青黛**：团队成员，急智型奇兵，屎袋破魅魔功的执行者。
  3. **雷东**：苏烬的羁绊型搭档，二次元团队组织者，负责提供民科理论和搞笑担当。
  4. **张行烈**：赛博朋克世界单兵武力巅峰，高科技装备提供者。
  5. **梁翠**：魅魔型反派，能力是魅惑，被屎袋破功后逃跑。

- **专属物品 / 技能 / 概念名词（前5个）**：
  1. **界桥**：连接诸天末日世界的权限系统。
  2. **气缚环**：苏烬的念力操控技能，可裹挟物体攻击。
  3. **空间戒指**：可收放水体、物体，用于制造领域战。
  4. **仿真面具**：异人制作的道具，可完美伪装身份。
  5. **归衡台**：跨世界任务监控设施。

## 可借鉴的操作手法
- **情节层**：
  1. **“掀桌”式转折**：在内部矛盾或外部压力积累到顶点时，主角用绝对暴力或规则制定权瞬间清场（操场枪决、手拧金属摆件），跳过所有程序正义。
  2. **争吵即通行证**：让角色的缺点或内讧在另一种筛选机制下成为优势（因下贱被合欢宗录取），制造反套路爽感。
  3. **规则前置加镣铐**：在主角使用金手指前，先通过对话详细交代限制条款，增强真实感和紧张感。
  4. **客户变投资人**：将任务对象转化为资源提供者，实现一举多得的经营爽感。
  5. **改头换面见旧人**：主角以新身份闯入旧日同伴的冲突现场，制造身份错位的戏剧张力。

- **风格层**：
  1. **严肃场景的屎尿屁解构**：用最不体面的道具（屎袋、裸奔）破解最体面的能力（魅惑），形成强烈反差笑点。
  2. **现代口语入侵异世界**：让角色在修仙/末日背景下使用“二逼学生”、“插队”、“通讯录”等词汇，制造荒诞感。
  3. **光影塑形法**：用光、影、材质描写来快速建立角色的气势和科技感（“被锋光裁出的影子”）。
  4. **“看傻子”视角切换**：先让读者通过其他角色视角知道陷阱布设，再切回反派视角展示其如何沾沾自喜地踩入陷阱。
  5. **说教式长句 + 断句式反应**：主角用逻辑严密的长句输出价值观，其他角色用短句、断句表现被碾压的震惊。

- **人物层**：
  1. **行动先于解释**：新角色切入时，先写其极端行为（闷头冲、丢出塑料袋），再揭示行为逻辑，用一次关键行动重新定义角色功能。
  2. **标签化快速定性反派**：通过配角之口给反派贴上功能性标签（“负责后勤”、“拉拢小团体”），不涉及深层动机，使其成为纯粹的“待清除符号”。
  3. **内心谴责与外部演讲的错位**：让窃听者内心痛骂主角无耻，同时主角正在发表“精通人性”的演讲，形成双重喜剧效果。
  4. **双主角互补拆台**：设计两个主角一个偏谋略一个偏直觉，在争吵中互相揭短又互相补充，争吵本身成为情节推进器。
  5. **能力插件式配角**：配角在战斗中主要作为“能力插件”存在（控水、科技侦查），力竭或撤出直接导致战局变化，功能明确。

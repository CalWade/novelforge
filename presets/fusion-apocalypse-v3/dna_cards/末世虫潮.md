# 末世虫潮 · DNA 卡

## 一句话定位
末世+全球神话幻想体降临+游戏化小队任务+主角靠吞噬进化与复活术建城封神。

## 戏剧结构
- 整书基调：**快节奏爽文，黑色幽默消解残酷，战斗密度高，情感直给不矫情。** 末世生存压力被金手指大幅对冲，核心是“变强-收队友-建势力-打BOSS”的升级循环，而非资源匮乏的挣扎。
- 典型的“5章情节单元”长什么样：
  - **起（1章）**：用日常/任务/突发事件快速拉升张力，明确本单元目标（救人/抢东西/守城/解谜）。
  - **承（1-2章）**：多线战斗或信息收集，展示敌我双方能力，制造“看似势均力敌”或“主角方陷入被动”的假象。
  - **转（1章）**：主角/队友爆种、援军天降、智力破局、金手指觉醒，瞬间扭转战局。
  - **合（1章）**：收割战利品、处理俘虏、复活死者、安顿民众，并用新悬念（新敌人/新地图/新阴谋）平滑过渡到下一单元。
- 信息释放节奏：
  - **即时解释**：新能力、新设定出现后，立刻通过旁白或角色对话解释清楚，不让读者长时间困惑。
  - **对话“讲述”为主**：大量关键信息（角色背景、任务目标、世界观设定）通过角色对话直接说出，而非场景“展示”。
  - **错位制造期待**：频繁使用“读者先于主角知道”（上帝视角展示敌人阴谋）或“主角先于读者知道”（古月感知到敌人但未说明）的信息差。
  - **扣留核心悬念**：敌人的幕后主使、完整战略意图、主角进化后的新能力，常被刻意保留到后续章节。
- 爽点频率：
  - **小爽点（每1-2章）**：战斗中的招式对轰、能力克制、队友毒舌吐槽、主角靠智力/直觉识破小阴谋。
  - **大爽点（每5章单元至少1个）**：绝对实力碾压（秒杀/群体复活）、援军天降逆转战局、伏笔回收式爆种（用早期设定碾压新敌人）、智力破局（识破敌方阳谋）。

## 写作风格
- 叙述视角 + POV策略：
  - **基础视角**：以主角古月为中心的有限第三人称。
  - **实际执行**：频繁切换为**全知上帝视角**，自由进入任何角色的内心（包括反派、龙套、新登场角色），用于解释设定、展示多线战场、制造信息错位。
  - **POV切换频率**：在同一章内可在3-5个角色视角间跳转，但始终以主角团为叙事重心。
- 句式 / 节奏 / 对白密度：
  - **句式**：战斗场景以短句和连续动作词为主（“古月瞬间消失，下一刻出现在远处”）；叙述和设定解释以中等长度简单复合句为主。
  - **节奏**：战斗时高速、密集；日常/过渡时明显放缓，形成张弛循环。
  - **对白密度**：**极高。** 对白承担了推进情节、解释设定、塑造人物、制造笑点、心理战挑衅等几乎所有功能。角色靠说话完成大部分叙事工作。
- 特色修辞 / 高频词 / 作者口味：
  - **招式名即修辞**：战斗的华丽感几乎完全依赖大量中二、具体的招式名（“辟地开天，夏舒版”、“雷扫八方”、“玄黄一气斩”）。
  - **网络梗与口语化**：大量使用现代流行语、网络梗和口语化内心独白（“靠”、“做女人挺好”、“厕所里点灯，找死”），消解严肃感。
  - **高频词**：“瞬间”、“恐怖”、“变态”、“直接”、“居然”、“吞噬”、“进化”、“秒杀”。
  - **作者口味**：偏爱**黑色幽默**处理悲剧（沈明变性）、**反差萌**角色（娘娘腔司徒越、天然呆安雪琪）、**数字和层级**明确的力量体系。
- 章末钩子的三种主力款式：
  1. **威胁升级/新危机浮现**：敌人变身/增援/新怪物登场，或主角中招/被困，以“命悬一线”或“更大的麻烦来了”作结。
  2. **新悬念/谜底浮现**：意外发现新物品/新地图/新角色，或主角产生顿悟但未说明内容，以“这到底是什么/他想到了什么”作结。
  3. **角色命运未卜**：角色硬接大招后生死不明，直接切断叙事。

## 人物 & 价值观
- 主角核心动机 + 性格关键词：
  - **核心动机**：**守护身边人（姐姐、队友）→ 报恩（希望城）→ 履行职责（救世主/城主）→ 获取信仰力变强。** 动机自私且真实，从个人情感逐步扩展到势力责任。
  - **性格关键词**：护短、冷静、有急智、该狠时绝不留情、对敌人狡诈（空城计）、对队友宽容。
- 主角的金手指是什么 + 作者如何避免主角“开挂无趣”：
  - **金手指**：吞噬/吸收能力（进化、获取敌人能力）、复活术、神级武器（葬神戟）、特殊体质（盘古原体晶石相关）、株连劫眼。
  - **避免无趣的手法**：
    1. **智力破局**：古月常靠观察、直觉、逻辑推理识破阴谋（如识破海族阳谋），而非纯力量碾压。
    2. **战术性撤退**：承认有打不过的对手，选择撤退而非强行开挂（如对海马皇）。
    3. **队友高光**：关键战斗的爽点常分配给队友（夏舒秒杀、宙斯觉醒），主角作为后盾和底牌。
    4. **代价与限制**：进化过程痛苦且有风险（变伪娘/真娘），复活术消耗巨大。
- 典型反派形象 + 反派的“真实威胁值”如何维持：
  - **典型形象**：神话幻想体（海格力斯、齐格菲尔德）、虫族（刀虫王、虫神）、海族（海马皇、死水王）、人类组织（神道）。
  - **威胁值维持手法**：
    1. **设定级压制**：引入“绝元体”、“禁法领域”等克制主角团常规手段的设定。
    2. **连环计/阳谋**：反派并非无脑冲锋，而是用计谋（声东击西、正面佯攻掩护阵法）。
    3. **阶段性胜利**：反派能在局部取得战果（斩下盘古拇指、重伤萧青、逼退古月），不是一直被碾压。
    4. **幕后黑手悬念**：真正的BOSS（如奥丁、冥海）长期不露面，通过手下和阴谋维持压迫感。
- 本书**价值观锚点**：读者被什么打动？
  - **爽感（绝对实力碾压、复仇、爆种）**
  - **友情与羁绊（队友间的信任、吐槽、牺牲与守护）**
  - **黑色幽默（用笑点消解悲剧，如沈明变性）**
  - **智力博弈（识破阴谋、将计就计）**
  - **成长与收获（进化、收队友、建城、受民众狂热崇拜）**

## 核心设定（**供“禁止直接抄袭”清单用**）
- 世界观公理：
  1. 末世由虫潮引发，同时全球神话中的幻想体（神、英雄、怪物）真实降临。
  2. 人类觉醒为新人类，拥有各种超能力，并可组成小队执行任务。
  3. 存在“演化物”这一特殊物品体系，能赋予能力但有弱点（如精神力被切断则失效）。
  4. 存在“信仰力”体系，主角可通过守护城市、展现神迹来获取。
  5. 存在“执念”设定，强烈的情感可残留于细胞/物体，影响现实。
- 主角专属设定：
  - **金手指来源**：吞噬/吸收能力，与盘古原体晶石血脉相连。
  - **身份秘密**：被认定为“救世主”，但本人只想守护身边人。
  - **真实立场**：不灭城城主，以信仰力为重要收益，但行动逻辑基于个人情感。
- 专属角色 / 组织 / 地名（前5个）：
  1. **古月**（主角）
  2. **夏舒**（主角队友，拥有“绝对领域”和“幻想体杀手”）
  3. **沈明**（主角队友，曾变性，希腊神话相关）
  4. **不灭城**（主角建立/守护的人类城市）
  5. **希望城**（主角最初所属的势力）
- 专属物品 / 技能 / 概念名词（前5个）：
  1. **葬神戟**（主角武器）
  2. **幻想体杀手**（夏舒的规则级匕首）
  3. **株连劫眼**（主角的范围秒杀技能）
  4. **演化物**（特殊物品体系总称）
  5. **绝元体**（免疫元素攻击的设定）

## 可借鉴的操作手法
- 情节层：
  1. **“螳螂捕蝉，黄雀在后”式连环计**：让多方势力围绕同一目标展开争夺，主角作为最后的“黄雀”得利，或识破自己也是“蝉”后反制。
  2. **“意外触发”式转折**：让角色的常规行为（如战斗余波、偷盗）意外触发新事件/新地图/新物品，将故事拉入新阶段。
  3. **“垃圾时间”填设定**：在主角进化/被困/赶路等情节空白期，通过角色对话或内心活动，高效抛出多个新设定。
  4. **“先行动后背景”塑造配角**：让新配角先在一场战斗中展现能力与性格，战斗结束后再通过回忆/对话补充其背景故事。
- 风格层：
  1. **招式名包装战斗**：为角色的每个技能/攻击动作起一个中二且具体的名字，在战斗中大声喊出，以此替代复杂的动作描写。
  2. **口语化内心独白制造笑点**：在紧张/残酷场景中，插入角色口语化、网络梗式的内心吐槽，瞬间消解严肃感。
  3. **“幸灾乐祸”式黑色幽默**：将角色的悲剧经历（如强者变性）通过其他角色的调侃和夸张反应，转化为读者的笑点和爽点。
  4. **“权威人士”定性新角色**：让一个已建立强大感的角色（如主角）用一句话揭示新角色的真实身份/实力，快速建立新角色的逼格。
- 人物层：
  1. **标签化快速切入**：新角色登场时，用一个极具特色的动作、一句口头禅或一个外貌特征（如兰花指、蛇发）瞬间立住人设。
  2. **“守护-失去-暴怒-挽回”情感流**：让主角因未能保护某人/物而暴怒，爆发出更强力量完成复仇，再立刻用金手指（如复活）挽回损失，完成情感闭环。
  3. **“对比升级”法介绍老对手**：当老对手再次登场时，明确写出其装备/实力/状态与上次的对比（如“从皮甲升级到精良铠甲”），快速体现威胁升级。

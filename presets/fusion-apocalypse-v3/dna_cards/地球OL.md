# 地球OL · DNA 卡

## 一句话定位
末世游戏降临，独行帝王以“召唤卡牌”与“吞噬万物”之力碾压全球，统一世界。

## 戏剧结构
- **整书基调**：快节奏、强碾压的末世争霸爽文。主角从不停歇，不是在战斗就是在收服新手下/获取新能力的路上。情绪基调是“帝王般的冷漠”与“绝对实力的碾压”，偶有对忠诚部下的温情，但整体保持一种超然的、玩家式的功利视角。
- **典型的“5章情节单元”长什么样（起-承-转-合的样板）**：
  - **起（第1章）**：主角进入新环境或遭遇新威胁，通过观察或对话快速摸清局势，确立目标（如获取某宝物、复仇、收服某人）。
  - **承（第2-3章）**：冲突与试探。敌方势力登场，展示能力并口出狂言，形成初步威胁。主角方则开始行动，或正面迎战，或暗中布局。此阶段常穿插配角视角，补充信息或制造悬念。
  - **转（第4章）**：实力展示与高潮。主角或主角方核心战力出手，用压倒性的金手指（召唤大军、吞噬能力、情报压制）瞬间击溃敌方最强点，形成“打脸”爽感。转折点往往是某个隐藏能力的揭示或某个计谋的成功。
  - **合（第5章）**：矛盾收束与新冲突开启。战后收尾，收编降将，治疗伤员，分配战利品。同时，章末必定埋下新的冲突引线（如更强的敌人出现、新的阴谋被揭露、某个神秘角色登场），直接钩向下一单元。
- **信息释放节奏（作者如何管理读者的好奇心）**：
  - **“先展示，后解释”**：主角的新能力或道具，总是在战斗中先展示其逆天效果，战后才通过对话或内心独白解释其原理和限制。
  - **“导游式”世界观交代**：通过新角色（如白汐羽、一团气）作为“导游”，将势力划分、规则、关键人物等大量设定信息一次性打包交代。
  - **“读者先于主角知道”的错位**：频繁使用。作者会短暂切入敌方或配角的视角，让读者知道某个阴谋或危机正在逼近，而主角尚不知情，从而制造持续的紧张感和期待感。
  - **“信息悬置”**：故意不解释某些物品的真实作用或角色的真实身份，将其作为长期钩子，在后续剧情中逐步揭晓。
- **爽点频率（每几章一个小爽点 / 大爽点）**：
  - **小爽点**：每1-2章一次。通常是主角方成员展示新技能、收服新手下、或者一次小规模冲突的快速胜利。
  - **大爽点**：每5章单元内至少1次。集中在“转”的部分，是主角亲自出手或核心金手指展示，形成对敌方最强者的碾压式秒杀，并引发全场震撼。

## 写作风格
- **叙述视角 + POV 策略**：以主角关雨的**单POV为主**，但频繁、短暂地切换到其他角色的视角（包括敌人、手下、旁观者），形成一种**“有限全知”视角**。切换主要用于三个目的：1. 补充主角视野外的信息；2. 通过旁观者的震惊来烘托主角的强大；3. 制造“读者先于主角知道”的悬念。
- **句式 / 节奏 / 对白密度（定量描述）**：
  - **句式**：战斗场景以**短句和连续动作**堆叠节奏，如“杀，一个不留”、“他双腿猛的一蹬地面，瞬间出现在关雨面前。”；日常和设定说明则用中长句。
  - **节奏**：极快。描写极少，多为功能性的动作和对话推进，形成“动作-反应-新动作”的高速循环。
  - **对白密度**：极高，占比约40%以上。情节推进、信息交代、人物塑造、冲突展现几乎全部依赖对白完成。
- **特色修辞 / 高频词 / 作者口味**：
  - **偏爱数字精确化**：用“4598”、“9527”、“敏捷破百万”等具体数字来制造冷酷、真实的游戏感。
  - **偏爱标签化词汇**：用“蝼蚁”、“变态”、“人渣”等词快速定性角色和实力。
  - **偏爱现代概念类比和反套路吐槽**：如“时间就像是女孩的丝袜”、“他的大脑简直就可以媲美鱼的大脑”，用网文特有的随意调侃制造间离效果和笑点。
  - **偏爱排比式列举**：在展示碾压态势时，常用“遇到XX，杀，遇到XX，杀”的排比句式。
- **章末钩子的三种主力款式**：
  1.  **未完攻击/对白型**：章末断在某个攻击即将命中、或某句极具冲击力的对白之后，制造即时冲突悬念。使用频率最高。
  2.  **威胁升级型**：通过展示新敌人的强大、或揭示主角未察觉的危机，让读者为主角处境担忧。
  3.  **谜底浮现型**：抛出一个新的谜团（如神秘道具、角色身份），激发读者的探究欲。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：初期是**复活托妮**，中后期转变为**统一全球、建立绝对秩序**的帝王式霸权。
  - **性格关键词**：**冷漠、务实、帝王心术、超然**。行为模式带有强烈的“玩家”色彩——以任务和利益为导向，用设置考验的方式观察人性，对敌人毫不留情，对部下则给予庇护和机会。
- **主角的金手指是什么 + 作者如何避免主角“开挂无趣”**：
  - **金手指**：1. **帝皇玩家**（一人一帝国，可召唤大军）；2. **神魔唤卡师**（可将万物封印为卡牌并召唤）；3. **百无禁忌**（可吞噬万物，包括能量、剧毒，并转化为己用）。
  - **避免无趣的手法**：1. **智力碾压**：主角经常靠情报差、心理战、机制理解来取胜，而非纯粹的力量对轰；2. **部下展示**：许多战斗由部下（如安倍晴明、封无、死帝）完成，展示团队成长和不同风格的战斗；3. **“先抑后扬”**：让主角先陷入看似被压制的困境，再通过揭示隐藏能力或计谋反转。
- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型形象**：**“装逼犯”型强者**。出场时能力展示华丽、口出狂言、气场十足，带有明显的邪魅或高傲特质（如德古拉伯爵、班猜、五天王）。
  - **威胁值维持**：1. **先建立威胁**：通过秒杀路人、展示逆天技能、或让主角方成员陷入苦战来建立压迫感；2. **快速打脸**：威胁建立后，在同一个单元内就被主角或主角方核心战力以更碾压的方式秒杀。威胁值是“一次性”的，用于制造单次爽点。
- **本书价值观锚点**：读者被什么打动？
  - **爽感**：最核心的锚点。来源于“绝对实力的碾压”、“智商压制的打脸”、“反派装逼被秒杀”的三段式结构。
  - **生存智慧**：主角的务实、功利、善于利用规则和人性，在末世中无往不利，提供了一种“强者思维”的爽感。
  - **忠诚与庇护**：主角对部下（如一号、零卫队）的庇护和给予机会，以及部下狂热的忠诚，提供了情感上的满足。

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1.  地球被一款名为“地球OL”的真实游戏系统覆盖，所有生物数据化。
  2.  游戏系统包含职业、技能、等级、副本、复活点等标准MMORPG元素。
  3.  存在多个服务器或“世界”（如天庭、魔法世界），玩家可通过特定方式穿越。
  4.  存在“帝皇玩家”这一特殊职业，可建立帝国，召唤NPC大军。
  5.  远古时代存在“太古四大魔人”（将臣、后卿、嬴勾、旱魃）等超规格存在。
- **主角专属设定**：
  - **金手指来源**：究极职业“神魔唤卡师”和“帝皇玩家”的双重身份。
  - **身份秘密**：来自“天外天”（现实世界）的玩家，知晓游戏本质。
  - **真实立场**：以“玩家”心态行事，目标是通关游戏、复活特定人物、最终统一所有世界。
- **专属角色 / 组织 / 地名（前5个）**：
  1.  **零之帝国 / 零始帝国**：主角建立的势力。
  2.  **天庭**：一个高级玩家聚集的服务器/区域，有六大势力。
  3.  **零卫队**：主角的狂信徒护卫队。
  4.  **安倍晴明**：主角的军师型部下，狡黠且擅长心理战。
  5.  **封无**：天庭“天道”公会会长，为守护天朝将自己改造为能量人。
- **专属物品 / 技能 / 概念名词（前5个）**：
  1.  **神魔唤卡师**：主角职业，可将万物封印为卡牌。
  2.  **百无禁忌**：主角技能，可吞噬万物。
  3.  **聚宝盆**：天庭的排行榜挑战副本。
  4.  **武神之心 / 天魔之心**：关键剧情道具，关联远古魔人。
  5.  **因果盗贼**：死帝的职业，能力与“结果预判”相关。

## 可借鉴的操作手法
**（这部分是融合新世界观时可直接借鉴的抽象技法，不含任何原著专属名词）**

- **情节层**：
  1.  **“导游式”信息释放**：在主角进入新地图时，立刻安排一个消息灵通的引导型配角，通过对话将势力格局、关键人物、基础规则等设定一次性打包交代。
  2.  **“先抑后扬”的爽点公式**：先让反派通过秒杀路人甲、展示华丽技能来建立威胁，再让主角用更碾压、更轻松的方式将其秒杀，并用一句淡定点评收尾。
  3.  **“智力反转”制造爽点**：让主角在看似公平的竞争或战斗中，利用对手不知道的“隐藏机制”或“信息差”轻松取胜，而非纯粹的力量对轰。
  4.  **“双线叙事”制造悬念**：在主角线进行的同时，切到配角或敌方视角，展示一个主角尚不知情的危机正在逼近，章末汇合，形成救援/冲突悬念。
  5.  **“时间跳跃”升华钩子**：在一个事件结束后，用“X年后”的时间跳跃快速收尾，并用一句颠覆前文认知的话（如“我用了十年来考验你”）开启新篇章。

- **风格层**：
  1.  **“数字精确化”制造冷酷感**：在描写实力、伤害、数量时，使用精确的数字（“敏捷破百万”、“三万人”），增强游戏感和真实感。
  2.  **“标签化词汇”快速定性**：用“蝼蚁”、“变态”、“枭雄”等高度概括的标签词，让新角色在出场瞬间就建立鲜明形象。
  3.  **“三拍子”战斗节奏**：战斗描写采用“技能名！”→“动作/特效描述”→“造成的结果”的三拍子结构，制造高速、清晰的战斗画面。
  4.  **“反套路吐槽”制造间离效果**：让主角或叙述者用现代思维、网络梗来评论玄幻/游戏世界的事件，形成幽默感和独特的叙述腔调。
  5.  **“排比式列举”渲染碾压感**：在展示压倒性力量或进行无差别攻击时，使用“遇到A，杀，遇到B，杀”的排比句式，强化气势。

- **人物层**：
  1.  **“冲突中登场”介绍新角色**：让新角色在战斗、骂战或绝境中登场，通过其应对冲突的方式（能力、台词、态度）快速完成性格定性。
  2.  **“内心独白”补充反派动机**：在反派攻击或落败时，插入一段简短的背景故事或内心独白（如“天生矮小被嘲笑”），快速建立其行为逻辑，让打脸更有情感重量。
  3.  **“旁观者震惊”烘托主角**：在主角展示实力后，立刻切到围观群众或敌人的视角，通过他们的震惊、恐惧、议论来侧面烘托主角的强大。
  4.  **“考验式”角色发展**：让主角设置一个针对人性弱点的幻境或考验，让配角在其中挣扎并做出选择，通过“修心”的胜利来完成角色的蜕变和获得力量。
  5.  **“标签化”团队构建**：为主角团队的核心成员赋予一个极端的、功能化的标签（如“军师”、“狂信徒”、“科学狂人”），让他们各司其职，减少主角的操作负担，丰富团队作战的层次感。
